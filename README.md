# Investorfyre


## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/shadwar123/Investorfyre.git
```
### 2. Set up Environment Variables from .env.example in server

### 3. Run backend
```bash
# In server folder
cd server
npm install
npm start

```
### 4. Run frontend
```bash
# In client folder
cd client
npm install
npm run dev
```
